# IP2Host
## Introduction
利用WebScan提供的API，通过IP反查域名
## Features
1. 实现日志功能
2. 实现自定义处理异常
3. 实现批量查询
4. 实现区分私有IP和公有IP
5. 实现简易进度条
## Installation
```bash
# 安装环境
pip install -r requirements.txt
```
## Usage
```bash
python3 main.py
```
